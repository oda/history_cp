# 総合評価

- 2007: 団体名（固有名詞）として成立（一次資料あり）
- 2008後半: ジャンル語として使用開始（複数独立例あり）
- 2009: 界隈内では一般名詞化が事実上完了
- 2011: UTPCがAtCoder上で開催（語の普及は既に進行済）
- 2012: AtCoder法人化（語の成立より後）

→ 「一般名詞化はAtCoder以前」という主張は十分に支持される。
→ AtCoderは語の“起源”ではなく“増幅器・制度化装置”と捉えるのが自然。
